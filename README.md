# Task Manager Assessment

 Full-stack Task Management Application built with NestJS (Backend) and ReactJS (Frontend).

## Prerequisites
- Node.js (v18 or higher)
- npm

## Installation & Running

### Backend
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm run start:dev
   ```
   The API will be available at `http://localhost:3005`.

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`.

## Features
- **Create Task**: Add a new task with a title and optional description.
- **List Tasks**: View all tasks with their status.
- **Update Status**: Change task status (OPEN -> IN_PROGRESS -> DONE).
- **Delete Task**: Remove a task permanently.
- **Responsive Design**: Clean UI built with Tailwind CSS.
